ℹ️ 关于
粉笔模考爬虫 mokao1.0 基于 Obsidian Web Clipper 模板设计理念。
使用 Handlebars 风格语法，支持 {{变量名}}、{{#each}} 循环和 {{#if}} 条件判断。

点击变量标签可快速插入。设置保存在浏览器本地。
📖 Obsidian 直存
运行 mokao_obsidian_server.py 后，扩展可将 Markdown 直接写入 Obsidian Vault，无需手动下载。

🖥 启动服务器
① 在终端运行（需要 Python 3.8+）：
cd /path/to/chrome-extension/
python mokao_obsidian_server.py --vault "你的Obsidian库路径"
② 首次运行会要求选择 Vault 目录（Windows 文件选择器）
③ 服务器运行后，在扩展设置中确认服务器连接正常即可
💡 原理：扩展通过 HTTP POST 将 Markdown 内容发送给本地 Python 服务器，服务器直接写入 Obsidian Vault 文件夹。
服务器只监听 localhost:8765，不暴露外网，安全可靠。vault 路径保存在本地配置文件，持久生效。
